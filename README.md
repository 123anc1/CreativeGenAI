🌟 CreativeGenAI

A Full-Stack AI Creative Generation System with Multi-Service Architecture
基于多服务架构的 AI 创意内容生成系统（Web + AI + Image Generation）

📌 项目简介

CreativeGenAI 是一个集 Web 前端 + 后端服务 + Python AI服务 + 图像生成模块 于一体的智能创意生成系统。

系统支持用户通过 Web 页面交互，调用 AI 模型生成创意内容，并进行结果展示与管理。

该项目用于展示 全栈开发能力 + AI工程集成能力 + 多服务架构设计能力。
